
trois informations doivent être fournies 

- sa position géographique dans l'espace : localisation 
- sa relation spatiale avec les autres objets spatiaux : topologie 
- ce qu'est l'objet avec un caractère d'identification ( code):  attribut 


le SIG permet de traiter les données spatiales et associées 

les données spatiales détermine les caractéristiques  spatiales d'une entité géographique où  sont représentés et identifiés tous les éléments graphiques : 

    	La localisation : coordonnées  par rapport a une echelle graphique de référence 
	    La forme: point, ligne, polygone 
	    La taille : longueur , périmètre  , surface 

- point : est désigné  pas ses coordonnées  et la dimension spatiale la plus petite 
- Ligne : a une dimensuin spatiale constitué d'une  succession de points proches les uns des autres  
- Polygone  : (Zone ou surface) est un élément de surface défini par une ligne fermée ou la ligne qui le délimite 

-------------------------------------------------------------------- 
Les données associés des entités géographiques permettent de compléter  la représentation géométrique de l'entité spatiale .  chaque élément de l'espace recois un code d'indentifications qui peut être numérique  ou littéral , ce code consititue en quelque sorte une étiquette caractérisant le ooint , la ligne ou le polygone 


il y trois catégorie de données  associés :  

- les données de classification  : ces données  permettent de ranger le point isolé , la ligne ouverte ou la ligne fermer  , dans un catégorie  
		- tout ce passe , comme si l'on distinguait plusieurs couche d'informations , que l'on  saisie successivement 
- Les donnés d'identification : Ces données permettent d'individualiser chaque objet figurant sur le plan ( nom propre de l'objet)  ex: numero de parcelle, numero de vanne  
- Le données attributaires  : Ces données viennent apporter une information supplémentaire  , propre a chaque objet identifié
