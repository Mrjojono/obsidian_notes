
**Audit**  

L'audit est l'enregistrement systématiques, chronologique et sécurisé  de toutes les actions pertinentes effectuées  dans la base de données 

 le résultat est  une **piste Audit (Audit trail)** 
L'audit est une fonction d'enregistrement gérer souvent par le SGBD lui même 

**Cryptage**  

Utiliser des technique de cryptage pour protéger  les données  au repos et en transis 

**Gestion des vulnérabilité**   

Identifier et corriger   les failles de sécurité   dans le système  de gestion de base de données (SGBD )  et les applications associées
