- L'intégrité des données ( cela veut dire que on doit retrouver les données dans les tables telles qu'ils on été laissé)
- cohérence  des données  ( exemple enregistrer  une données avec des valeur qui n'existe)

La **cohérence des données** désigne le fait que les **informations stockées dans un système** (comme une base de données) **restent correctes, exactes et logiquement compatibles entre elles**, quelle que soit l’opération effectuée (ajout, modification, suppression, etc.).

La cohérence des données garantit que **toutes les données racontent la même “vérité”** dans tout le système.
