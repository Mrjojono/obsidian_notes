
Les bases de Données OLAP (Online Analytical Processing) qui contiennent des informations statistiques afin d’être capables d’extraire les informations sous forme de cube multidimensionnel dans un but d’aide à la décision par exemple. Les statistiques contenues dans les bases de données OLAP s’appuient sur des informations contenues dans une base de données OLTP

==Lorsque SQL Server est installé pour gérer des bases de données OLAP, on dit de SQL Server qu’il est un serveur décisionnel ou Data Warehouse==
